Umpire Coder — Settings

Tag Types

These appear as buttons in the overlay. Drag to reorder is not yet supported — delete and re-add to reorder.

OBS Connection

First use: OBS → Tools → WebSocket Server Settings → Enable WebSocket server. Set a password and enter it below.

In OBS, add a Window Capture source to your scene pointing at Chrome (Sources → + → Window Capture → select your Chrome window). The extension only controls Start/Stop recording — the source stays as-is.

Leave blank to use the default install location (C:\Program Files\obs-studio on Windows, /Applications/OBS.app on Mac).

Output Location

OBS recordings and processed clips & PDF reports are both saved here.

Recording Quality

When a match starts, the recording is automatically cropped to the video player area. This value controls how much larger than the raw <video> element an ancestor can be before it is considered a page layout wrapper rather than part of the player. Increase it if overlays (scoreboards, graphics) are being cut off; decrease it if unrelated page content is being included.
